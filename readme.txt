== Description ==

Display Most popular posts or most viewed posts on your blog using widget in sidebar.
Plugin offers a wide range of settings to customize the layout of your widget.
Plugin also supports custom post types

Layout Options

* select post type 
* give number of posts to show
* show/hide thumbnail
* thumbnail position top/left
* show/hide post author
* show/hide post date
* show/hide post tags
* show/hide post categories
* show/hide post number of views
* show/hide number of views
* editable VIEWS text

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Popular Posts to manage layout.
4. Go to Appearance->Widgets , use WP-xPerst Popular Posts Widget
